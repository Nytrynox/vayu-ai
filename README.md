# VAYU AI

AI-powered hyperlocal pollution death risk predictor with 72-hour AQI forecasting, NASA fire detection integration, and personalized health risk scoring.

## Features

- 72-hour AQI forecasting for any location
- NASA FIRMS fire detection data integration
- Personalized health risk scoring based on medical history
- Carbon footprint calculator
- Hospital capacity predictor during high-pollution events
- Interactive Streamlit dashboard

## Tech Stack

Python, Streamlit, NASA APIs, Scikit-learn, Plotly

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/vayu-ai.git
cd vayu-ai
pip install -r requirements.txt
```

### Usage

```bash
streamlit run app.py
```
